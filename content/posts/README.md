# 환영합니다!

이 폴더의 모든 노트는 자동으로 블로그에 퍼블리시됩니다.

## 사용 방법

1. 이 `publish` 폴더에 마크다운 파일을 작성하세요
2. 터미널에서 `./sync-obsidian.sh` 실행
3. `hugo server` 로컬 확인 또는 `fly deploy` 배포
