# 커버 이미지 사용하기

이 글은 커버 이미지가 어떻게 표시되는지 보여주는 예제입니다.

## 커버 이미지 추가 방법

글의 front matter에 다음과 같이 추가하면 됩니다:

```yaml
cover:
  image: /images/cover.PNG
  alt: "이미지 설명"
  caption: "이미지 캡션"
  relative: false
```

이제 모든 글에 이렇게 귀여운 커버 이미지를 추가할 수 있습니다!
